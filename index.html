Program 2: Sum of digits

Algorithm

1.Take input from the user
2.Initialize a variable "sum" to 0
3.Loop through each digit in the input number
4.Add each digit to the "sum"
5.Print the "sum"

Solution
 
num = int(input("Enter a number: "))
sum = 0
while num > 0:
    digit = num % 10
    sum += digit
    num //= 10
print("Sum of digits:", sum)

Program 4: Check if Palindrome

Algorithm

Take input from the user
1.Initialize two variables "start" and "end" to the beginning and end of the string
2.Loop through each character in the string, comparing the characters at "start" and "end"
3.If the characters are not the same, print "Not a palindrome" and exit the loop
4.If the loop completes without finding any non-matching characters, print "Palindrome"

Solution
 
string = input("Enter a string: ")
start = 0
end = len(string) - 1         
while start < end:
    if string[start] != string[end]:
       print("Not a palindrome")
       break
    start += 1
    end -= 1
else:
    print("Palindrome")

Program 7: GCD (Greatest Common Divisor)

Algorithm

Take input from the user for two numbers
Find the minimum of the two numbers
Loop through each number from 1 to the minimum
Check if both numbers are divisible by the current number in the loop
If yes, update the value of "gcd" to the current number

Solution
 
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
         
min_num = min(num1, num2)
         
gcd = 1
         
for i in range(1, min_num+1):
    if num1 % i == 0 and num2 % i == 0:
        gcd = i
         
print("GCD:", gcd)

Program 9: Armstrong Number

Algorithm

Take input from the user
Initialize a variable "sum" to 0
Loop through each digit in the input number
Calculate the cube of each digit and add it to the "sum"
If the "sum" is equal to the input number, print "Armstrong number", else print "Not an Armstrong number"

Solution
 
num = int(input("Enter a number: "))

sum = 0
temp = num
            
while temp > 0:
    digit = temp % 10
    sum += digit ** 3
    temp //= 10
            
    if num == sum:
       print("Armstrong number")
    else:
       print("Not an Armstrong number")

Program 10: Count Vowels

Algorithm

Take input from the user
Initialize a variable "count" to 0
Loop through each character in the input string
If the character is a vowel, increment the "count" variable
Print the value of "count"

Solution

vowels = "aeiouAEIOU"

Program 10: The Final Pattern

Required Pattern

4*3*2*1
12*11*10*9
8*7*6*5
16*15*14*13

Solution
 
n = int(input("Enter the number of rows: "))
num = 1
for i in range(n):
    if i % 2 == 0:
        for j in range(num, num+4):
            print(j, end="")
            if j != num+3:
                print("*", end="")
        print()
        num += 4
    else:
        for j in range(num+3, num-1, -1):
            print(j, end="")
            if j != num:
                print("*", end="")
        print()
        num += 4
